# Kafka Analytics Playground - Project Overview

## Purpose
A hands-on Kafka learning project simulating a digital storefront (Steam-like game store) with real-time analytics. The project demonstrates event-driven architecture patterns using Kafka for stream processing.

## Project Type
Educational Python + Kafka project for learning stream processing and real-time analytics.

## Key Features
- Real-time event generation for game store interactions (views, wishlist, cart, purchases, searches)
- Stream processing with Kafka consumers
- Real-time analytics dashboard with FastAPI + SSE
- 5 event types across 6 Kafka topics
- Conversion funnel tracking (view → wishlist → cart → purchase)
- Revenue and trending games metrics

## Phase
Currently in Phase 1 (Store Analytics). Future phases planned for gameplay tracking and microservices.

## Development Status
Active learning/playground project with working producers, consumers, and dashboard infrastructure.
